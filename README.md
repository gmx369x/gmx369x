### Principles

- **Async by default** -- if it touches I/O, it's async
- **Fail gracefully** -- primary > fallback > rule-based default, the system never crashes
- **Token-conscious** -- every LLM call has a budget, never burn tokens on what code can solve
- **Containers are truth** -- the running container is the source of truth
